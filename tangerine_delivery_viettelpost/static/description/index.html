I. Getting started

Install module Grab Express Shipping

II. Configuration Grab shipping method

In this section, we will proceed to set the parameters and connect to Grab Server to get the access token. This access token will help authenticate API connections with the Grab Server.

Step 1: Go to: Inventory/Configuration/Delivery/Shipping Methods/Grab Express/Grab Express Configuration Tab

Step 2: Input:

  • PartnerID
  • ClientID
  • Client Secret

Tips

Click on button to check the connection status to the Grab server

Step 3: Click on button

Warning

Access token lifespan is 7 days by default which means request once and reuse 7 days.

Finally, if successful, the access token will be saved, along with an expiration date.

III. Delivery cost billing automated workflow

In this section, we will configure properties to help automate and optimize the delivery process. This is very necessary, it helps users limit manual operations that take a lot of time during operation.

Step 1: Go to: Settings/Configuration/Delivery/Shipping Methods/Grab Express/Grab Express Configuration Tab

Step 2: Activate attribute

Automatically Confirm Purchase Order

After creating a delivery order, the system will automatically create a purchase order corresponding to the order code with the status as draft. If the "Automatically Confirm Purchase Order" attribute is enabled, the system will automatically confirm the purchase order.

Automatically Create Bill

Next, after automatically creating a purchase order, if the "Automatically Create Bill" attribute is enabled, the system will automatically create an invoice based on the previously created purchase order. This billing is understood as a billing to cover delivery costs if the seller pays.

Automatically Confirm Bill

Next, after automatically creating a billing, if the "Automatically Confirm Bill" attribute is enabled, the system will automatically confirm the automatically created invoice.

Automatically Register Payment

Finally, after automatically confirm a billing, if the "Automatically Register Payment" attribute is enabled, the system will automatically register payment for delivery order.

Grab Express delivery address convention

Please follow the conventions given, in case you do not follow this convention, you may get a system error.

For Vietnam Country:

  • The addresses will be gathered from 4 levels. Street, Ward, District, Province. Not input Country
  • Only pass the level name “L1: Province” as in link. For example: Valid: Ho Chi Minh. Invalid: Thanh pho Ho Chi Minh.

Example address valid: 18 duong so 19, P.8, Q.Go Vap, Ho Chi Minh

Grab Express phone number convention

Please follow the conventions given, in case you do not follow this convention, you may get a system error.

For Vietnam Country:

  • Phone number must start with country code without +. (E.164 standard)
  • Reference link

Example phone number valid: +84812212228

Other countries

For countries where Grab Express has other support. Please follow the convention between your country and the grab provider.

V. Estimation cost delivery

After configuring the shipping addresses for your customers. In this section, we will estimate the shipping costs for your sales order.

Step 1: Go to: Sales, Create a new sale order

Step 2: Click on button to open pop up Add a shipping method

Step 3: Choose Shipping Method

Step 4: Choose Service Type (Default: Instant)

Step 5: Choose Vehicle Type (Default: Bike)

Step 6: Click on button to get estimated cost from Grab Express provider

Step 7: Click on button to confirm add cost to the sale order

Finally, You have successfully estimated the shipping costs for your sales order

VI. Creation delivery order

Once the cost has been estimated and confirmed by your customer. We will proceed to create a delivery order with the Grab Express provider.

Step 1: Go to: Inventory/Delivery Orders/Choose Delivery Order/Additional Info Tab
